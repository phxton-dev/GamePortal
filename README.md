# Game Portal

GamePortal is a collection of games and programs that allows users to run multiple interactive applications from a single entry point.

## Dependencies
To run the GamePortal, you need the Processing `core.jar` in your `lib` folder. 

## Games Available

### 1. ASCII Art
A program where you can view random ASCII art. Displays different ASCII symbols and artwork each time you play.

### 2. BuzzFeed Quiz
A personality quiz that determines which Minecraft PvP kit you are based on your answers to questions about your personality. Results are tracked to show the most popular category across all players.

### 3. Blackjack
A casino card game where you try to profit by playing Blackjack. Start with $1000 and manage your bets strategically to maximize your winnings. High scores track your best profit.

## High Scores

The Game Portal automatically tracks high scores for each game in `Highscore.csv`:
- **ASCII Art**: No scoring
- **BuzzFeed Quiz**: Tracks the most popular Minecraft PvP kit category
- **Blackjack**: Tracks your highest profit
